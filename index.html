RolePlayGateway's API

We currently provide read-only data through our REST API at api.roleplaygateway.com, providing a window into the resources on RolePlayGateway as JSON-encoded objects.

All endpoints are served from the following location, utilizing a resource to determine what you are accessing:

URI: http://api.roleplaygateway.com/v1/{resource}

Pagination

All results can be paginated using the GET parameters limit and (start or page).

Resources

Users

Index

Provides a list of users.

URI: http://api.roleplaygateway.com/v1/users

Entity

Serves the details about a specific user ID.

URI: http://api.roleplaygateway.com/v1/users/{ID}

Roleplays

Index

Provides a list of roleplays.

URI: http://api.roleplaygateway.com/v1/roleplays

Entity

Serves the details about a specific roleplay ID.

URI: http://api.roleplaygateway.com/v1/roleplays/{ID}

Aspects

Serves the list of aspect about a specific roleplay ID.

URI: http://api.roleplaygateway.com/v1/roleplays/{ID}/{aspect}
Groups
List
URI: http://api.roleplaygateway.com/v1/roleplays/{ID}/groups
Detail
URI: http://api.roleplaygateway.com/v1/roleplays/{ID}/groups/{groupID}
Places
List
URI: http://api.roleplaygateway.com/v1/roleplays/{ID}/places
Detail
URI: http://api.roleplaygateway.com/v1/roleplays/{ID}/places/{characterID}
Characters
List
URI: http://api.roleplaygateway.com/v1/roleplays/{ID}/characters
Detail
URI: http://api.roleplaygateway.com/v1/roleplays/{ID}/characters/{characterID}
